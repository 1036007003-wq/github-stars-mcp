GitHub has **100M+ developers**. Your repo's star count is social proof. But growing from 0→100 stars requires data-driven strategy. This MCP server gives your AI assistant the data it needs to help you grow.

## ⚡ Quick Start (30 seconds)

```bash
git clone https://github.com/1036007003-wq/github-stars-mcp.git
cd github-stars-mcp
npm install
```

Add to your MCP client:

```json
{
  "mcpServers": {
    "github-stars": {
      "command": "node",
      "args": ["/path/to/github-stars-mcp/index.js"],
      "env": {
        "GITHUB_TOKEN": "your-token"
      }
    }
  }
}
```

**No token needed** for public repos (60 req/hr). Add a token for 5,000 req/hr.
